You've already used **some** plenty of times — *some milk, some apples, some money*. Today comes its negative twin: **any**. The split is simple. **Some** lives in **positive** sentences. **Any** lives in **negatives** and most **questions**. There's one big exception — *Would you like some …?* and *Can I have some …?* — when you're offering or asking for something, **some** comes back even in a question. Plus their cousins: *somebody/anybody, something/anything*.

## Part C — *some* and *any* without a noun

Same trick as *mine* (lesson 61): drop the noun if the listener already knows it.

- I didn't take any pictures, but Jessica took **some**. *(= some pictures)*
- You can have some coffee, but I don't want **any**. *(= any coffee)*
- I've just made some coffee. Would you like **some**? *(= some coffee)*
- 'Where's your luggage?' 'I don't have **any**.' *(= any luggage)*
- 'Are there any biscuits?' 'Yes, there are **some** in the kitchen.' *(= some biscuits)*

## Now you try

Write **six** sentences using *some* or *any*.

1. *I bought ___ … today.* (positive — some)
2. *I didn't buy ___ … .* (negative — any)
3. *Are there ___ … in the fridge?* (question — any)
4. *Would you like ___ … ?* (offer — some)
5. *Can I have ___ … , please?* (request — some)
6. *I saw ___ at the bus stop.* (somebody / anybody?)

<Callout type="tip" title="Star challenge — flip the polarity">
  Take any positive *some*-sentence and rewrite it three ways: as a negative (with *any*), as a question (with *any*), and as an offer (with *some*). Example: *I have some bread.* → *I don't have any bread.* / *Do you have any bread?* / *Would you like some bread?*
</Callout>

In English, repeating the noun twice sounds clumsy: *I need a pen. Do you have a pen?* The fix is **one** — a tiny stand-in word that means *a + (the noun we just said)*. Plural? Use **ones**. Today you also learn the combos: *this one, that one, the one near …, the brown one, a clean one, some new ones*.

<Callout type="tip" title="One = a + noun">
  *Would you like **one**?* = *Would you like **a chocolate**?* The same trick works with most singular countable nouns: *I need a pen — do you have **one**?*
</Callout>

## Part A — *one* = *a / an + noun*

- I need **a pen**. Do you have **one**? *(one = a pen)*
- A: Is there **a bank** near here? <br /> B: Yes, there's **one** at the end of this street. *(one = a bank)*

  *another* = *an + other*. So *another **one*** = *another (chocolate, biscuit, cup …)*. *That biscuit was nice. I'm going to have **another one**.*
</Callout>

## Now you try

Write **six** sentences using *one* or *ones*.

1. *I need a pen. Do you have ___ ?*
2. *I don't like these shoes. I like the ___ .* (colour or other adj. + ones)
3. *That cup is dirty. Can I have ___ ?* (a + adj + one)
4. *Which book? — The ___ .* (the one + where/who)
5. *These biscuits are nice. I'll have another ___ .*
6. *Those flowers are beautiful. Which ___ ?* (one or ones?)

<Callout type="tip" title="Star challenge — never repeat the noun">
  Take a paragraph you wrote yesterday and underline each noun that appears more than once. Try replacing the second mention with *one / ones / it / them*. The text gets shorter and sounds more natural every time.
</Callout>

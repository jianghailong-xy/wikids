In English we use three little words to say what something **is** or who someone **is**: **am**, **is**, and **are**. Let's learn them with our friend Lisa.

## Meet Lisa

Lisa says hello! Read what she tells you about herself:

- "My name **is** Lisa."
- "I **am** 22."
- "I **am** American. I **am** from Chicago."
- "I **am** a student."
- "My favourite colour **is** blue."
- "My favourite sports **are** football and swimming."
- "I **am** interested in art."
- "I **am not** interested in politics."

<Callout type="tip" title="Did you spot them?">
  Find every **am**, **is**, and **are** in Lisa's sentences. How many did you count?
</Callout>

## Short forms — how "I am" becomes "I'm"

When we speak, we squeeze two words into one. We **drop a letter** and put an **apostrophe (')** in its place. Watch how it happens:

    { drop: " " },
    { swap: "a", to: "'" },
    { keep: "m" },
  ]}
/>

The same trick works for **is** and **are**:

<Contraction
  caption="he is → he's"
  segments={[
    { keep: "he" },
    { drop: " " },
    { swap: "i", to: "'" },

  ]}
/>

<Contraction
  caption="they are → they're"
  segments={[

    { drop: " " },

/>

<Callout type="tip" title="The pattern">
  Step 1: take away the **space**. &nbsp; Step 2: take away **one letter** of am/is/are. &nbsp; Step 3: put an **apostrophe** where it disappeared.
</Callout>

## Saying "no" — the negative

To say **no**, we add **not** after am / is / are.

- I **am not** late.
- He **is not** here.
- They **are not** tired.

The same shrinking trick works on **is not** and **are not** — watch:

<Contraction
  caption="is not → isn't"
  segments={[
    { keep: "is" },
    { drop: " " },
    { keep: "n" },
    { swap: "o", to: "'" },
    { keep: "t" },
  ]}
/>

<Contraction
  caption="are not → aren't"
  segments={[
    { keep: "are" },
    { drop: " " },
    { keep: "n" },
    { swap: "o", to: "'" },
    { keep: "t" },
  ]}
/>

<Callout type="info" title="Two ways are okay">
  **He is not** can shrink **two** ways: **he isn't** *or* **he's not**. Both are correct — pick the one that feels easy to say.
</Callout>

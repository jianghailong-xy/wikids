## The big rule

Every subject has its own form of **to be**. Watch the arrows: each pronoun on the left points to **the only verb it can use**.

<PronounVerbMap />

So how does this look in a real sentence? Watch the subject change — the verb has to follow:

<Callout type="info" title="Why three forms?">
  English picks the verb based on **who** is doing it. **I** uses **am**, one person/thing uses **is**, and a group (or **you**) uses **are**.
</Callout>

## Short forms — how "I am" becomes "I'm"

When we speak, we squeeze two words into one. We **drop a letter** and put an **apostrophe (')** in its place. Watch how it happens:

<Contraction
  caption="I am → I'm"
  segments={[
    { keep: "I" },
    { drop: " " },
    { swap: "a", to: "'" },
    { keep: "m" },
  ]}
/>

The same trick works for **is** and **are**:

<Contraction
  caption="he is → he's"
  segments={[
    { keep: "he" },
    { drop: " " },
    { swap: "i", to: "'" },
    { keep: "s" },
  ]}
/>

<Contraction
  caption="they are → they're"
  segments={[
    { keep: "they" },
    { drop: " " },
    { swap: "a", to: "'" },
    { keep: "re" },
  ]}
/>

<Callout type="tip" title="The pattern">
  Step 1: take away the **space**. &nbsp; Step 2: take away **one letter** of am/is/are. &nbsp; Step 3: put an **apostrophe** where it disappeared.
</Callout>

## Saying "no" — the negative

To say **no**, we add **not** after am / is / are.

- I **am not** late. <SayIt text="I am not late." iconOnly />
- He **is not** here. <SayIt text="He is not here." iconOnly />
- They **are not** tired. <SayIt text="They are not tired." iconOnly />

The same shrinking trick works on **is not** and **are not** — watch:

<Contraction
  caption="is not → isn't"
  segments={[
    { keep: "is" },
    { drop: " " },
    { keep: "n" },
    { swap: "o", to: "'" },
    { keep: "t" },
  ]}
/>

<Contraction
  caption="are not → aren't"
  segments={[
    { keep: "are" },
    { drop: " " },
    { keep: "n" },
    { swap: "o", to: "'" },
    { keep: "t" },
  ]}
/>

<Callout type="info" title="Two ways are okay">
  **He is not** can shrink **two** ways: **he isn't** *or* **he's not**. Both are correct — pick the one that feels easy to say.
</Callout>

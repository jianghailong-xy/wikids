You already know **there is** and **there are** for talking about what exists *right now*: *There **is** a train every hour. There **are** 550 rooms.* Today we put that little **there**-sentence into a time machine. We'll send it **back** (*there was, there were*), into the **just-now** (*there has been, there have been*), and **forward** (*there will be*).

## Part A — there was / there were (the past)

When the *thing* you're talking about was around **earlier** — last night, yesterday, ten minutes ago — use **there was** for one and **there were** for many. It's the past partner of **there is / there are**.

<Callout type="info" title="Same shape, just one tense back">
  **there is** → **there was** &nbsp;·&nbsp; **there are** → **there were** <br />
  **there isn't** → **there wasn't** &nbsp;·&nbsp; **there aren't** → **there weren't** <br />
  **is there ...?** → **was there ...?** &nbsp;·&nbsp; **are there ...?** → **were there ...?**
</Callout>

Watch the same idea travel from **today** to **last night**:

| now (present) | then (past) |
|---|---|
| **There is** nothing on TV tonight. | **There was** nothing on TV last night. |
| **There are** 550 rooms in this hotel. | **There were** 550 rooms in that hotel. |
| **Are there** any problems? | **Were there** any problems yesterday? |
| **There isn't** anything to eat. | **There wasn't** anything to eat. |

<SayIt text="There was nothing on TV last night." /> &nbsp; <SayIt text="There were 550 rooms." /> &nbsp; <SayIt text="Were there any problems?" />

A handful of everyday examples:

- I was hungry when I got home, but **there wasn't** anything to eat. <SayIt text="I was hungry when I got home, but there wasn't anything to eat." iconOnly />
- We stayed at a very big hotel. **There were** 550 rooms. <SayIt text="We stayed at a very big hotel. There were 550 rooms." iconOnly />
- '**Was there** a swimming pool?' 'Yes, **there was** a beautiful one.' <SayIt text="Was there a swimming pool? Yes, there was a beautiful one." iconOnly />

<Callout type="warning" title="It / they vs there">
  Don't mix **there was** up with **it was** or **they were**. *There* tells us **what existed**; *it / they* names a **specific thing**. <br />
  ✅ **There was** a clock on the wall. **It was** broken. <br />
  ❌ It was a clock on the wall. *(no — use **there was** when you're introducing the clock)*
</Callout>

## Part B — there has been / there have been (past → now)

You met the present perfect in Lessons 15–18: a past action that **still touches now**. The same trick works with *there*: **there has been** (one thing) and **there have been** (many things) say something has happened, and we can see the result *now*.

### The trap: there was vs there has been

These two look almost the same, but English is fussy. The rule from Lesson 20 still holds: **a finished time word** (*last night, yesterday, ten minutes ago*) demands the **past** — not the present perfect.

| ✅ past simple (with finished time) | ❌ present perfect |
|---|---|
| **There was** an accident **last night**. | There has been an accident last night. |
| **There were** lots of people at the party **on Saturday**. | There have been lots of people at the party on Saturday. |
| ✅ present perfect (no finished time) | |
| Look! **There's been** an accident. | |
| **There have been** a lot of accidents recently. | |

<Callout type="info" title="One short test">
  Is there a **finished time word** in the sentence (*yesterday, last night, in 2010, two hours ago*)? <br />
  → **Yes** → use **there was / there were**. <br />
  → **No** (and the result still touches now) → use **there has been / there have been**.
</Callout>

## Part C — there will be (the future)

To talk about something that will exist **later** — tomorrow, next week, in five minutes — use **there will be**. One shape for everything: it doesn't change for *one* or *many*.

## All forms in one table

A single home for the whole family — keep this table close.

| Time | One thing | Many things | Negative | Question |
|---|---|---|---|---|
| **now** | there **is** | there **are** | there isn't / aren't | is there? / are there? |
| **past** | there **was** | there **were** | there wasn't / weren't | was there? / were there? |
| **past → now** | there **has been** | there **have been** | there hasn't / haven't been | has there been? / have there been? |
| **future** | there **will be** | there **will be** | there won't be | will there be? |
